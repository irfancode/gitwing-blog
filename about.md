# About GitWing

GitWing is an open-source project exploring the future of Git tools on macOS.

## The Mission

Build the fastest, most native Git experience for Apple Silicon Macs.

## Why Open Source

Great Git tools should be free. This blog documents the journey of building one.

## Tech Stack

- **SwiftUI** — Native UI framework
- **Apple Silicon** — Optimized for M1/M2/M3
- **GitHub Pages** — Free hosting with Jekyll
